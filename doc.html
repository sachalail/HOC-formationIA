📁 Arborescence & Emplacement des fichiers

Voici comment sont organisés les dossiers sur le serveur GitHub Pages :

├── index.html ← Page d'accueil / Hub principal
├── doc.html ← La présente documentation
├── css/
│ └── main.css ← Feuillets de styles centralisés (HOC)
└── documents/
    └── journees-formation/
        ├── niv1 et 2/
        │ └── index.html ← Dashboard Planning Niv 1 & 2
        └── niv3 et 4/
            └── index.html ← Dashboard Planning Niv 3 & 4

✍️ Comment ajouter ou modifier du contenu ?

  • 1 Ouvrir le dépôt GitHub
    Rendez-vous sur le dépôt via le bouton en haut de page ou directement depuis GitHub.
  • 2 Pour ajouter un nouveau fichier / tableau de bord
    Allez dans le dossier documents/journees-formation/, créez un nouveau dossier (ex: niv5/) puis un fichier index.html à l'intérieur.
  • 3 Lier le CSS centralisé
    Assurez-vous que votre nouveau fichier HTML importe correctement le CSS commun en ajustant le chemin relatif dans le <head> :
    <link rel="stylesheet" href="../../../css/main.css">
  • 4 Mettre à jour le Hub principal
    Pour que la nouvelle page soit accessible, éditez index.html à la racine et ajoutez une nouvelle carte (<a class="card" href="...">) pointant vers votre nouveau fichier.
  • 5 Valider (Commit)
    Cliquez sur Commit changes... en haut à droite sur GitHub. La modification sera déployée en 1 à 2 minutes sur le site en ligne !
